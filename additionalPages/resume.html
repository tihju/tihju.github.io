Generalized Knapsack Problem

This project compare the runtime of N-Tupe implementation and subset implementation to solve Generalized Knapsack Problem in brute force. This project is to help understand NP problem. Analyzing data--in this case, time different of two implementation to understand the behavior of n-tuple and subset sum implementation. Due to the brute force implementation, the project only gether limited data set. Thus, it is difficult to see the significant time different of these two implementations. Used Java, Shell script.
